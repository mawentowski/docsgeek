Base URL

The base URL is the root URL of the API. It includes the domain and the version number. In the case of the Steritas REST API, the base URL is https://service.steritas.com/api/**version_number**, substituting **version_number\*\* with the currently deployed API version.

Domain: https://service.steritas.com

Version: /api/**version_number**
